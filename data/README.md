# Política de datos

Los datos brutos de mercado no se redistribuyen en este repositorio.

El proyecto utiliza series históricas OHLCV obtenidas mediante yfinance / Yahoo Finance y noticias recuperadas mediante GDELT. El repositorio incluye notebooks, código y outputs demostrativos seleccionados para documentar la trazabilidad metodológica del Trabajo Fin de Máster.

Cuando corresponde, los archivos incluidos son datasets derivados, tablas, figuras o salidas experimentales necesarias para explicar el flujo de trabajo. Para una reproducción completa, los datos de entrada deben reconstruirse ejecutando los notebooks y scripts conforme a las fuentes indicadas en la memoria.

No se incluyen claves API, credenciales, contraseñas ni datos personales.
